Mulai Menjelajah

Router

Network Layer (L3)

Otak dari jaringan yang menghubungkan segmen berbeda. Menggunakan IP Address untuk menentukan rute terbaik paket data menuju internet atau jaringan lain.

Switch

Data Link Layer (L2)

Manajer lalu lintas lokal. Switch mendistribusikan data hanya ke perangkat tujuan menggunakan MAC Address, menjaga jaringan tetap efisien.

Access Point

Wireless (802.11)

Jembatan tanpa kabel. Mengubah sinyal kabel menjadi gelombang radio, memungkinkan kita terhubung ke jaringan tanpa batasan fisik kabel LAN.
